Scientific Knowledge Brokering Portal Solr Proxy

This web application should be accessible to the search interface

Configure solr host settings in %userhome%/.unccd/config.json

Default configuration is: 

{
    "solrhost": "http://localhost:8983/solr/",
    "collection": "unccd/",
    "handlers":["select", "partner","partnerdetail","roe","kb","cbw","selectmap", "partnermap","roemap","kbmap","cbwmap"]
}


URL for this proxy is configured in property "url" in config.js in the search interface:

var config = {
    "url": "http://localhost:8080/solrproxy/fw",
    "collectionName": "unccd",
    "searchHandler": "select",
    "highlightsDir": "highlights",

    "autocompleteHandler": "terms"
}
